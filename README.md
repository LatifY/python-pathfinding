# Python Pathfinding

### Requirements
```Python
pip install pathfinding
pip install pyfiglet (Alternative)
```

### Usage
This Python program is finding shortest path on a rectangle matrix between 2 location that you input.

For Example:
```
0 = Obstacle  
1 = Way       

Matrix Width: 3
Matrix Height: 3
Type a matrix(3-3)
1 0 1
1 0 1
0 1 0
Start Position: 0 0
End Position: 2 0

Path Length:  5     
********************
(0, 0)
(0, 1)
(1, 2)
(2, 1)
(2, 0)
********************
Operations:  5
+---+
|s#e|
|x#x|
|#x#|
+---+ 
```
